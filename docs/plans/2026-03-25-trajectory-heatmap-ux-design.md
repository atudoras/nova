## 2. Trajectory Plot Redesign (`plot_pca_trajectories_general`)

### 2.1 New `color_by` parameter

```
color_by = c("group", "Treatment")   # default "group" preserves existing behaviour
```

| `color_by`    | Line / point colour       | End-point text label       |
|---------------|--------------------------|----------------------------|
| `"group"`     | Treatment Ă— Genotype combo (current) | last timepoint value |
| `"Treatment"` | Treatment only            | **Genotype name** via `ggrepel::geom_text_repel` |

Implementation note: when `color_by = "Treatment"` the `scale_color_manual`
key maps **treatment** names (not group_id combos) to colours.  The genotype
label is pulled from the last row per `(group_id, well_id)` for individual
plots, or per `group_id` for averaged plots.

### 2.2 Start / end markers

Replace the current `"B"` plain-text start label:

- **Start point** â†’ open diamond `shape = 18` (or `shape = 5`), white fill,
  coloured border; nudged label `"tâ‚€"` (or actual first timepoint string).
- **End point** â†’ filled circle `shape = 21`, black fill; label = genotype
  (when `color_by = "Treatment"`) or last timepoint value (current default).

Both geom layers already exist (`first_last_points`, `first_last_combined`);
only the `shape`, `fill`, and `label` columns need updating.

### 2.3 Timepoint subtitle

After computing `group_average_trajectories`, extract the ordered unique
timepoint values and append to `labs()`:


Apply to **all** trajectory sub-plots (individual, averaged, combined).

### 2.4 ggrepel dependency

Add to `DESCRIPTION` Imports: `ggrepel (>= 0.9.0)`.
Add `@importFrom ggrepel geom_text_repel` in the roxygen block.

## 3. Heatmap Combination Plot (`create_mea_heatmaps_enhanced`)

### 3.1 Trigger

`split_by = "combination"` (new value; existing `"Treatment"` / `"Genotype"`
values unchanged).

### 3.2 Behaviour

1. Create a `combo_label` column: `paste(Treatment, Genotype, sep = "_")`.
2. Sort rows by `combo_label` (so same-treatment wells cluster).
3. Pass two annotation bars to `pheatmap`: one for Treatment, one for Genotype
   (two separate named colour lists).
4. Title: `"MEA Heatmap â€” Treatment Ă— Genotype"`.
5. Filename: `<prefix>_heatmap_combination.png`.
